# 📊 Student Performance Analyzer (NumPy Project)

## 📌 Project Overview

The **Student Performance Analyzer** is a data analysis project built using **NumPy and Python**.
The goal of this project is to analyze student academic performance using numerical operations and vectorized computations.

This project demonstrates how **NumPy can be used for fast statistical analysis** on structured datasets.

---

## 🎯 Objectives

The project analyzes student marks to identify:

* Student average performance
* Top-performing students
* Subject-wise performance
* Hardest subject for students
* Pass/Fail statistics
* Student ranking
* Grade classification using vectorized operations

## 🔍 Analysis Performed

### 1️⃣ Student Average Score

Calculate the average score of each student.

### 2️⃣ Top Performer

Identify the student with the highest average score.

### 3️⃣ Subject-wise Average

Compute the average score for each subject.

### 4️⃣ Hardest Subject

Find the subject with the lowest average score.

### 5️⃣ Pass / Fail Analysis

Determine how many students passed or failed.

### 6️⃣ Student Ranking

Rank students based on their average scores.

### 7️⃣ Grade Classification

Assign grades using a vectorized grading system.

| Score | Grade |
| ----- | ----- |
| 90+   | A     |
| 80–89 | B     |
| 70–79 | C     |
| 60–69 | D     |
| <60   | F     |
